■ A-1: 1年高リターンモメンタム
  ─────────────────────────────
  [リターン]
  [必] 1年リターン >= 15%（年換算）
  [必] 直近3ヶ月リターン >= 3%（短期モメンタム継続）
  [必] 1年リターン >= ベンチマーク（カテゴリ平均）
  [推] 1年リターン >= 25%
  [除] 最大ドローダウン(1年) <= -30%（過度なリスク）
  [コスト]
  [必] 信託報酬 <= 1.0%（コスト許容上限）
  [フィルター]
  [必] AUM（純資産） >= 100億円（国内）/ 1億USD（海外）
  [必] 設定後年数 >= 1年（運用実績確保）

■ A-2: 長期安定成長トレンド
  ─────────────────────────────
  [リターン]
  [必] 3年年率リターン >= 10%
  [必] 5年年率リターン >= 8%（存在する場合）
  [必] 直近1年リターン > 0%（下落トレンド除外）
  [推] 3年年率リターン >= 15%
  [推] 5年年率リターン >= 12%
  [リスク]
  [必] 3年年率ボラティリティ <= 25%（過度なリスク除外）
  [フィルター]
  [必] AUM >= 500億円（国内）/ 5億USD（海外）
  [必] 設定後年数 >= 3年

  [推] カテゴリ内ランキング 上位10%

  [必] RSI(14) >= 45 かつ <= 75
  [フィルター]
  [必] AUM >= 100億円 / 1億USD

 スコアリング基準（長期投資重視）
==================================================

【総合評価スコア計算式】
  基礎スコア: マッチ戦略数 × 1.0 ポイント
  グロース系戦略（A-1/A-2/F-1/G-1/H-1/H-2）: × 2.0 ポイント（スコア2倍）
  安定系戦略（B-2/B-3/C-1/C-2/G-2）: × 1.5 ポイント

  加点要素:
  1年リターン >= 30%: +5点
  1年リターン >= 20%: +3点
  1年リターン >= 12%: +2点
  1年リターン >= 8%: +1点
  信託報酬 <= 0.1%: +4点
  信託報酬 <= 0.2%: +3点
  信託報酬 <= 0.5%: +1点
  AUM >= 1兆円 / 100億USD: +3点
  AUM >= 5,000億円 / 50億USD: +2点
  AUM >= 1,000億円 / 10億USD: +1点
  シャープレシオ >= 1.5: +3点
  シャープレシオ >= 1.0: +2点
  シャープレシオ >= 0.8: +1点
  新NISA対応: +2点

【ハイリスク除外基準】
  RSI > 80（過熱域）
  最大ドローダウン <= -40%（高ドローダウン）
  年率ボラティリティ > 35%（超高ボラ）
  信託報酬 > 2.0%（高コスト過剰）
